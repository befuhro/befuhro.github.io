Wipe stack effect

This section aims to highlight a product in different colors. By stacking tags on top of each other and progressively unmask the tag that contains a product's content (based on scroll progression), we will obtain this effect.

The parameters available to customize this section are:

  • Length of the scrolling effect (in pixel)
  • Font size for desktop and for mobile
  • Font color (for each product tag)
  • Background color (for each product tag)
  • Image (for each product tag)

Upsell inside ajax cart

This upsell is implemented inside the scrolling part of an ajax cart. It aims to be noticed by the customer without being to intrusive. The products from the collection selected in the parameters are available by clicking on the arrows. And the product can be added on the cart by clicking on the button 'add to cart'.

The parameters available to customize this section are:

  • Collection of product to display
  • Font and decoration color
  • Background color
  • Enable the upsell

Feedback section

This section was made to show stylish customer review on a website. It's highly customizable, the merchant can change the rating of the review and stars will reflect the score given.

The parameters available to customize this section are:

  • Title
  • Overall rating
  • Color of review's background, filled stars, empty stars, arrow and arrow's background
  • Author's name and review (for each review)
  • Rating for each review

Dynamic badges on footer

This section was also made to add some trust to the brand of the shopify's seller. Unlike the other one, this one is implemented on the footer of the theme, so it appears on every page of the website. It is also animated, the user of the theme can choose the type of the animation (it can be either continuous or jerky like the one on the demo).

The parameters available to customize this section are:

  • Text and image (for each block)
  • Font color
  • Background color
  • Duration of the animation
